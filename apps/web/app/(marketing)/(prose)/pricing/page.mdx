# Pricing

Ligala is free to use during early access. We're not taking a platform fee on
payments today — every peso billed through a Ligala invoice settles directly to the
lawyer's PayMongo or PayPal merchant account.

## Clients

| Feature | Cost |
| --- | --- |
| Find a verified lawyer | Free |
| Sign engagement letters | Free |
| Pay invoices via PayMongo or PayPal | Provider fees only |
| Apply discount codes | Free |
| File pro bono requests | Free |

## Lawyers

| Feature | Cost |
| --- | --- |
| Public profile + directory listing | Free |
| IDMeta KYC verification | Free during early access |
| Run paid and pro bono cases | Free |
| Issue invoices and accept payments | Free (provider fees apply) |
| Issue your own discount codes | Free |
| Receive referrals and share-link signups | Free |

## When this changes

When we introduce a platform fee or a paid tier, we'll announce it at least 60 days
in advance, every existing lawyer keeps the early-access terms on existing cases,
and we'll publish the new schedule on this page.
